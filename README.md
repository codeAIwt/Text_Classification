
## 环境要求
1. 安装Python 3.12
2. 创建并激活虚拟环境：
   ```bash
   python -m venv venv
   venv\Scripts\activate  # Windows
   ```
3. 安装依赖库：
   ```bash
   pip install -r requirement.txt
   ```

## 数据集说明
数据集位于`邮件_files/`目录下，包含多个以`.txt`为扩展名的文本文件。文件名格式为`[标签].txt`，其中标签为整数类型。

例如：
- `0.txt`：标签为0的邮件
- `1.txt`：标签为1的邮件

## 模型介绍
项目使用中文BERT预训练模型`bert-base-chinese`进行文本分类任务。模型架构包括：
- BERT编码器：用于提取文本特征
- 分类器层：用于将特征映射到分类标签

模型训练过程中会自动保存最佳模型权重和最终模型权重。

## 使用方法

### 运行程序
```bash
python Pre-trainedTextClassification.py
```

### 程序流程
1. 加载邮件文本和标签数据
2. 划分训练集和测试集
3. 创建数据加载器
4. 加载BERT模型
5. 检查是否存在已保存的模型
   - 如果存在，直接加载并评估
   - 如果不存在，开始训练模型
6. 训练过程中自动保存最佳模型和最终模型
7. 评估模型性能并输出分类报告

## 代码说明

### 主要功能模块

1. **数据加载**：`load_data_from_txt()`函数从文件加载邮件文本和标签
2. **数据预处理**：`create_dataloaders()`函数创建数据加载器，处理文本编码
3. **模型训练**：`train_model()`函数实现完整的训练流程
4. **模型评估**：`eval_model()`和`get_predictions()`函数用于评估模型性能
5. **模型保存与加载**：自动保存最佳模型和最终模型，支持加载已训练模型

### 关键参数配置
```python
RANDOM_SEED = 42          # 随机种子
BATCH_SIZE = 16           # 批处理大小
NUM_EPOCHS = 3            # 训练轮数
LEARNING_RATE = 2e-5      # 学习率
MODEL_SAVE_PATH = './model/best_model_state.bin'  # 模型保存路径
```

## 运行结果
程序运行后会输出以下信息：
1. 数据加载情况
2. 训练集和测试集划分情况
3. 模型加载信息
4. 训练过程（如果未找到已保存模型）
5. 模型评估结果（分类报告）

## 注意事项

1. **模型保存**：程序会自动保存两个模型文件：
   - `best_model_state.bin`：训练过程中性能最佳的模型
   - `best_model_state_final.bin`：训练结束时的最终模型

2. **模型加载**：程序会优先加载最佳模型，如果不存在则加载最终模型

3. **训练时间**：训练时间取决于数据集大小和硬件配置，建议在GPU环境下运行以提高速度

4. **数据格式**：确保邮件文件命名格式正确（`[标签].txt`），且文件编码为UTF-8
